# Quickstart: Introduction to Physical AI Module

## generating Content

1. Use the `robotics-course-writer` agent to draft each chapter.
   ```bash
   # Example prompt
   Draft the 'Foundations of Physical AI' chapter for Module 5.
   ```

2. Review the generated content against the learning objectives.

## Integration

1. Use the `docusaurus-architect` agent to integrate the new files.
   ```bash
   # Example prompt
   Add the new Physical AI module files to the Docusaurus structure and update the sidebar.
   ```

## Deployment

1. Use the `docusaurus-deployment` skill to deploy the changes.
   ```bash
   skill: "docusaurus-deployment"
   ```
